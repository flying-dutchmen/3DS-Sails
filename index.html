3DS Sails

A native "true-colour" Canvas for the Nintendo 3DS platform.

versatile get & set for both pixel or rectangular regions.

Draw Text, Lines, Boxes, Polygons, Circles & tilted Ellipses

Seamless alpha bytes. No calling separate functions for... It's done automatically

"Elegant" use of left, right, bottom & ... gfxFramebuffer's as your blank canvas.

All source Code with exception of "Dual_for" macro presented has undergone debugging & passed Eg. is functional/usable
